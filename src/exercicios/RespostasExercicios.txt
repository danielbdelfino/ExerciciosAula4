1)O que é classe coesa?
R: Classe que possui uma unica responsabilidade.

2)Como fazer para detectar classes não coesas?
R: Classes que possuem mais de uma responsabilidade.

3)O que fazer com classes não coesas?
R: Refatorar para que as mesmas possuam uma única responsabilidade.
